Random Password Generator

**Автор:** [Регина Байдрахманова]

Описание

Графическое приложение для генерации случайных паролей с возможностью настройки параметров и сохранением истории. Реализовано на Python с использованием библиотеки `tkinter`.

Функциональность

- Настройка длины пароля (4–32 символа)
- Выбор типов символов: цифры, буквы, спецсимволы
- Генерация криптографически стойких паролей
- Сохранение истории в JSON-файл
- Отображение истории в табличном виде
